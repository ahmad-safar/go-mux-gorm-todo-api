# go-mux-gorm-todo-api

This is a simple REST API built with Go, Mux and GORM.

## Installation

```bash
git clone https://github.com/varsfr/go-mux-gorm-todo-api.git
```

## Usage

```bash
cd go-mux-gorm-todo-api
go mod tidy

go run main.go
```
